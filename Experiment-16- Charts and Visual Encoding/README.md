## 📘 Introduction

Data visualization is the process of **representing data in graphical form** such as charts and graphs. It helps in understanding patterns, trends, and relationships in data easily.

Different types of charts are used depending on the type of data and analysis required. Python provides powerful libraries like **Matplotlib and Seaborn** for creating visualizations.

Visual encoding refers to the use of visual elements like **color, size, shape, and position** to represent data effectively.

---

## 📚 Theory

Data visualization is an important part of data analysis as it helps convert complex data into an easy-to-understand format.

**Line charts** are used to show trends over time. They are useful for analyzing changes in data such as study hours or sales over days.

**Bar charts** are used for comparison between different categories. They represent data using rectangular bars with lengths proportional to values.

**Histograms** are used to show the distribution of numerical data. They divide data into intervals (bins) and display frequency.

**Scatter plots** are used to show relationships between two variables. They help identify correlations such as study hours vs marks.

**Advanced visualizations** include adding labels, legends, colors, and gridlines to make charts more informative.

**Seaborn library** provides better and more attractive visualizations compared to Matplotlib with less code.

These visualization techniques help in **better data interpretation and decision making**.
